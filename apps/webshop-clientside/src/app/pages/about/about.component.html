About page

Ik ben Thimo Luijsterburg (2184240). Dit is mijn angular webshop project voor periode 2.1 van Avans.

De casus voor dit project heb ik de keuze gemaakt om een webshop te maken met het angular framework. In de section hieronder is de ERD van dit project te zien. Hierin is te zien dat er producten op de webshop komen die reviews kunnen hebben en besteld kunnen worden door een gebruiker. Dit houd in dat er ook authenticatie in de applicatie zal komen. Daarnaast krijgt een gebruiker op basis van goed gereviewde producten ook aanbevelingen voor nieuwe producten die andere gebruikers met hetzelfde aankoop patroon goed gereviewd hebben. Daarnaast kan een gebruiker zijn bestelde orders inzien met een overzicht van totale uitgaven en een percentage btw hiervan.

Use cases

  1. Een user kan inloggen en registreren
  2. Een ingelogde user kan producten bestellen.
  3. Een ingelogde user kan een review op een product achterlaten
  4. Een user krijgt aanbevolen producten op basis van andere gebruikers die hetzelfde producten goed gereviewd hebben.
  5. Een product heeft een fabrikant
  6. Een ingelogde user kan zijn totale uitgave zien
  7. Een ingelogde user totale btw van de gekochte producten zien
  8. Een ingelogde user kan een overzicht van zijn reviews zien
